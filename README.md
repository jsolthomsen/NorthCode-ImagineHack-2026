# NorthCode-ImagineHack-2026
This is the application for North Code at ImagineHack 2026. Our project is called Water Vision.

Team members: Jeppe Solvang Thomsen, Munira Dirie, Ugbaad Mohamed.

Technologies used: soilgrids API (https://soilgrids.org/), OpenWeather Map Pollution API, Weatherbit API, Java (Spring Boot), React, JavaScript, CSS.

Challenge and approach: We picked track 2 regarding secure and energy-aware cloud. We chose to go for an important part of construction as our use case with finding water in low-resource areas. This is what led us to make Water Vision.
